Setting up the Titan skin and menus

There are a few ways to integrate the available media items the addon makes available into the home screen, one way is to use a Kodi Skin that allows you to set up custom menu items and select video addon targets, this means you can set up a customised menu to allow you view at a glance all your new and interesting media. The Kodi skin Titan is a great example of this.

To set up the Titan Kodi skin:

  • Install EmbyCon addon
  • Install Titan skin
  • Set menu to single row (Optional)
  • Turn on "smart shortcuts" for favourties in Titan skin settings
  • Add favourites for EmbyCon menu items (press the 'C' key to bring up the items context menu)
  • Restart Kodi
  • Add new menu items to Titan
  • Add Widgets to the menu items
