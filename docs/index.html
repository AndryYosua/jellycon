What is EmbyCon?

EmbyCon is an addon for Kodi that allows you to browse and play back your Movie, TV Shows and Music from your Emby media server using Kodi.

For more information on Emby check out the official Emby Media Server site.

Installing EmbyCon

You can install the EmbyCon Addon from the official Kodi addon repository.

  • Open Kodi
  • Select Add-on from the main menu
  • Select "Download" at the bottom of the Add-on menu
  • Select "Video add-ons"
  • Select "EmbyCon" from the video add-on list
  • Select "Install"

Home Screen Integration

Due to the fact this addon does not populate the built in Kodi database none of the built in home screen widgets work by default. This is because Kodi does not know about your content.

There are a few ways to fix this, one way is to use a Kodi Skin that allows you to set up custom menu items. Another way is to edit the skin home screen file directly.

There are examples of both of these approaches in the menu on the left.
