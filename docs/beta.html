Beta Versions

If you want to try the beta test versions you can download the following beta repository zip and in Kodi under Addons select install from ZIP and install the Beta Repository.

repository.beta.emby.kodi-1.0.5.zip

Once the Beta Repository is installed in Kodi you can select Install addon from repository and select the "Emby Beta Repository", under video add-ons select EmbyCon.
